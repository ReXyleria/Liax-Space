# UI Tokens

This package owns the shared visual tokens for Liax Space.

The page background token is `#faf9f5`; the product does not use photo backgrounds, paper image backgrounds, blue-purple gradients, or high-saturation technology blue.

Brand buttons use `#c96442` with `#faf9f5`. This pairing is warm and on-brand, but its contrast is weak for ordinary small text. Brand button text should be larger or heavier, and small-text situations need a higher-contrast accessible alternative.
